###### 2. Write the `correct answer` from the following options and give an explanation (2-5 lines).

```javascript
function sum(a, b) {
  return a + b;
}

sum(1, "2");
```

- A: `NaN`
- B: `TypeError`
- C: `"12"`
- D: `3`

<details><summary><b>Answer</b></summary>
<p>

#### Answer: C: "12"

<i>Explanation: In JavaScript, when the + operator is used with a string and a non-string (in this case, a number), JavaScript implicitly converts the non-string operand to a string and concatenates them. So, "1" (implicit conversion of 1 to a string) + "2" results in the string "12".</i>

</p>
</details>

###### 4. Write the `correct answer` from the following options and give an explanation (2-5 lines).

```javascript
function sayHi(name) {
  return `Hi there, ${name}`;
}

console.log(sayHi());
```

- A: `Hi there,`
- B: `Hi there, undefined`
- C: `Hi there, null`
- D: `ReferenceError`

<details><summary><b>Answer</b></summary>
<p>

#### Answer: B: Hi there, undefined

<i>Explanation: The sayHi function expects a parameter name, but when it's called with sayHi() in the console.log statement, no argument is provided. As a result, the name parameter inside the function is undefined. The string template then includes this undefined value, resulting in "Hi there, undefined" being logged to the console.</i>

</p>
</details>

###### 5. Write the `correct answer` from the following options and give an explanation (2-5 lines).

```javascript
let count = 0;
const nums = [0, 1, 2, 3];

nums.forEach((num) => {
  if (num) count += 1;
});

console.log(count);
```

- A: 1
- B: 2
- C: 3
- D: 4

<details><summary><b>Answer</b></summary>
<p>

#### Answer: C: 3

<i>Explanation: The forEach method iterates over each element in the nums array. The condition if (num) checks if the current element is truthy (not equal to 0). Since all elements in the array except 0 are truthy, the count is incremented for each non-zero element. Therefore, the final value of count is 3.</i>

</p>
</details>
